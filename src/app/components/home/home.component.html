Descripción del Proyecto

Esta aplicación web fue desarrollada utilizando Angular y la librería de componentes PrimeNG. El sistema implementa una autenticación simple, con un login y manejo de roles.

Se incluye una guarda que protege rutas para usuarios no logueados, y se simula un sistema de autenticación con JWT mediante un interceptor.

Características

  • La aplicación obtiene todos los datos desde una API sugerida por la empresa, usada a modo de prueba.
  • Se ha añadido paginación, filtrado y búsqueda en todas las tablas de datos.
  • El sistema cuenta con notificaciones para todas las operaciones realizadas.

Entidades del Sistema

El sistema está compuesto por varias entidades que se relacionan entre sí:

  • Usuarios
  • Posteos
  • Comentarios
  • Álbumes
  • Fotos
  • To-Do's

Se trabajó en la implementación de un CRUD completo para estas entidades, respetando las relaciones entre ellas.
